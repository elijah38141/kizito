# Library Management System

Plain PHP, MySQL, HTML, and library management system with Admin and Librarian roles.

## Features

- User authentication for Admin and Librarian
- Admin dashboard
- Manage librarians
- Manage books
- Register members
- Issue and return books
- Automatic fine calculation
- Search books
- Daily and overall reports
- System settings
- Database backup and restore

## Setup

1. Create a MySQL database:

```sql
CREATE DATABASE library_system CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

2. Import the schema:

```bash
mysql -u elijah -p library_system < database/schema.sql
```

3. Update database credentials in `config.php`.

4. Start PHP locally:

```bash
php -S 127.0.0.1:8010
```

5. Open `http://127.0.0.1:8010`.
